# 神之书 (神の書)
- 在无尽的汉化工作中渐渐意识到日语之路学习不完，仅仅靠自学的`N2`是完全不太够的，所以我就想能不能开发一个工具辅助自己进行工作，这样既可以提升开发效率，也可以不断完善这个工具最终也许可以成为一个全面的日语学习工具
- 得益于先现代Ai技术的如此发达，我只需要进行API的调用即可，暂时也不需要LangChain进行定制模型的训练(<s>主要是我也不会</s>)所以这个项目就出现了，暂定计划是
***
1. 一期
   1. 设计出前端界面，提供上传漫画的功能
   2. 后端接受，然后通过OCR(<s>该用哪个</s>)想办法将文字识别出来
   3. 清洗，处理
   4. 保存
2. 二期
   1. 链接大模型，将识别的数据发送给他
   2. 清洗返回的数据
   3. 返送给前端
3. 现阶段先这样吧
